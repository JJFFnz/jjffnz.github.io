HTML Table of Elements

Elements Definition of Elements
<nav> This is the area where you can insert the navigation panel. Make sure to have it after the head section of the HTML
<body> This is where the viewable content of the website will sit
<h1>through to <h6> These are elements that define headings. h1 is more important (therefore larger) than h6
<section> Creates a section that isn't 'seeable' but helps to 'section' off various areas of a website, such as chapters, articles, etc
<div> This element, similar to section, is for dividing sections of the code but it is more used to apply custom styles to the group of HTML elements
